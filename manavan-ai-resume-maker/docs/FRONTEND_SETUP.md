# Frontend Setup

The frontend is built using **React + Vite + Tailwind CSS**.

---

## Step 1 Install Dependencies

Navigate to the frontend folder.

```bash
cd frontend
```

Install packages.

```bash
npm install
```

---

## Step 2 Start Development Server

Run the frontend.

```bash
npm run dev
```

The application will start at:

```
http://localhost:5174
```

---

## Step 3 Verify UI

Ensure the following components are working:

- Navbar
- Resume Upload
- Template Selection
- Resume Analysis Dashboard
