# Features

## Resume Upload

Users can upload resumes in:

- PDF
- DOCX

---

## AI Resume Analysis

The system extracts resume text and performs analysis.

---

## Resume Score

Provides an overall score indicating resume quality.

---

## Skill Detection

Identifies skills present in the resume.

---

## Missing Skills

Suggests important skills missing from the resume.

---

## ATS Compatibility

Evaluates whether the resume can pass ATS screening systems.

---

## Suggestions

Provides recommendations to improve the resume.
